EC2 Streamlit 배포 데모 영상 대본

1. 시작 및 제출자 소개

안녕하세요. 정보융합학부 2023204020 김우현입니다.
이번 영상에서는 Streamlit 앱을 AWS Learner Lab EC2 환경에 배포하고, 브라우저에서 EC2 주소로 접속해 실제로 실행되는 모습을 보여드리겠습니다.

오늘 시연에서 중요한 점은 브라우저 화면만 보여주는 것이 아니라, 앱을 조작하는 동안 EC2 터미널의 로그가 함께 출력되는 장면을 같이 보여드리는 것입니다.

2. EC2 주소 확인

먼저 오른쪽 터미널은 EC2에 SSH로 접속한 화면입니다.
현재 EC2의 퍼블릭 IP를 확인하겠습니다.

명령어:
curl ifconfig.me

이 주소를 브라우저 주소창에 사용해서 접속할 예정입니다.

3. Streamlit 프로세스 확인

이제 Streamlit 앱이 EC2에서 실행 중인지 확인하겠습니다.

명령어:
ps -ef | grep streamlit

현재 앱은 main.py를 기준으로 8501 포트에서 실행되고 있습니다.
실행 명령은 다음과 같은 형태입니다.

nohup venv/bin/python -m streamlit run main.py --server.port 8501 --server.address 0.0.0.0 > streamlit.log 2>&1 &

4. 로그 실시간 확인 준비

이제 EC2 터미널에서 로그 파일을 실시간으로 확인하겠습니다.

명령어:
tail -f streamlit.log

이 상태에서 브라우저에서 앱을 조작하면, 오른쪽 터미널에 login_success, answer_changed, result_submit_success 같은 로그가 출력됩니다.

5. 브라우저에서 EC2 Streamlit 접속

왼쪽 브라우저 주소창에 EC2 주소와 Streamlit 포트 번호를 입력합니다.

http://<EC2_PUBLIC_IP>:8501

이 화면은 Streamlit Cloud가 아니라 EC2 퍼블릭 IP로 접속한 화면입니다.
앱 제목은 "광운대역 도착 통학 생존 퀴즈"이고, 첫 화면에는 학부, 학번, 이름이 표시되어 있습니다.

6. 로그인 시연 및 로그 확인

기본 계정으로 로그인하겠습니다.

아이디: student
비밀번호: 1234

로그인 버튼을 누르면 브라우저는 퀴즈 화면으로 이동하고, 오른쪽 EC2 터미널에는 login_success 로그가 출력됩니다.
비밀번호는 로그에 남기지 않도록 처리했습니다.

7. 퀴즈 조작 및 입력 로그 확인

이제 퀴즈 문항을 조작해 보겠습니다.
객관식 답을 선택하고, 단답형 입력창에 답을 입력하고, 슬라이더도 움직여 보겠습니다.

제가 답을 선택하거나 입력할 때마다 오른쪽 터미널에는 answer_changed 로그가 출력됩니다.
로그에는 사용자명, 문항 번호, 문항 ID, 답안 유형, 입력 길이만 표시됩니다.
답안 원문은 기록하지 않습니다.

8. 결과 보기 및 로그 확인

모든 문항에 답한 뒤 결과 보기 버튼을 누르겠습니다.

결과가 계산되면 앱 화면에는 총점과 지역별 이해도 그래프가 표시됩니다.
동시에 EC2 터미널에는 result_submit_success 로그가 출력되고, 로그 안에는 사용자명과 점수가 함께 표시됩니다.

만약 답하지 않은 문항이 있는 상태에서 결과 보기를 누르면 result_submit_blocked 로그가 출력되어 미답변 문항 수도 확인할 수 있습니다.

9. 추가 조작 로그 확인

결과 창 열기, 다시 풀기, 퀴즈 데이터 캐시 새로고침, 로그아웃 같은 버튼도 조작해 보겠습니다.

각 조작에 따라 오른쪽 터미널에 다음 로그가 출력됩니다.

- result_dialog_open
- quiz_retry
- quiz_cache_refresh
- logout

따라서 영상에서는 사용자가 Streamlit 앱을 조작하는 장면과, EC2 터미널에서 그 조작에 대응하는 로그가 출력되는 장면을 동시에 확인할 수 있습니다.

10. 마무리

정리하면, 이 프로젝트는 EC2 환경에서 Streamlit 앱을 실행하고, 브라우저에서 EC2 퍼블릭 IP로 접속할 수 있도록 배포했습니다.
또한 앱 조작 로그를 streamlit.log에 남기도록 구현했기 때문에, 영상에서 브라우저 조작과 EC2 터미널 로그를 함께 확인할 수 있습니다.

최종 제출물은 GitHub public repository 주소와 YouTube 일부 공개 영상 링크입니다.
이상으로 EC2 Streamlit 배포 시연을 마치겠습니다.
